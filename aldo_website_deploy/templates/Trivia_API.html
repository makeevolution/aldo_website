Trivia API

Concepts used: API scraping
Applications used: Python, Flask, jQuery, HTML

This small side project is done for the purpose of understanding API scraping. The questions and possible answers are taken from a Trivia API provider. For each button click, data is taken from the API, transformed into JSON format and jQuery is used to randomise the position of the correct answer everytime a new question is loaded. Any suggestions on how to improve this small widget is appreciated.
